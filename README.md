Agentic AI: Real-Time Weather & Web Search Agent

An AI agent that retrieves **real-time weather data** from [Tomorrow.io](https://www.tomorrow.io/) and searches live information from the web using [DuckDuckGo](https://duckduckgo.com/). 
Built with **DeepSeek-R1-Distill-LLaMA-70B**, **LangChain**, and **LangGraph**, this project demonstrates an autonomous agent that can fetch, process, and present information intelligently.
